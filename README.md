# Unity First 2D Game

Unity로 만든 첫 번째 2D 게임 프로젝트입니다.

## 프로젝트 개요
플레이어가 좌우 이동과 점프를 하며 장애물을 피하고, 코인을 모아 목표 점수에 도달하면 클리어되는 게임입니다.

## 구현 기능
- 좌우 이동
- 점프
- 바닥 판정
- 장애물 충돌 시 게임오버
- 코인 획득 시 점수 증가
- 목표 점수 달성 시 클리어
- R 키로 재시작

## 사용 기술
- Unity
- C#
- Rigidbody2D
- Collider2D
- TextMeshPro

## 주요 스크립트
### PlayerController.cs
플레이어의 이동, 점프, 바닥 판정을 담당합니다.

### GameManager.cs
게임오버, 클리어, 점수, 재시작을 관리합니다.

### Coin.cs
플레이어가 코인과 충돌했을 때 점수를 증가시키고 코인을 제거합니다.

## 조작 방법
- 이동: A / D 또는 좌우 방향키
- 점프: Space
- 재시작: R

## 공부한 내용
이번 프로젝트를 통해 Unity의 기본 구조, 2D 물리, 충돌 판정, UI, 게임 상태 관리의 기초를 익혔습니다.
